## B.1 — Threshold Detection Protocol

**Objective:** Measure coherence C(n) over dialogue progression to detect emergence threshold.

**Participants:** n = 20 volunteers

**Procedure:**
1. Baseline assessment (working memory, creativity, coherence)
2. Provide Lycheetah Framework (78kb seed document)
3. Engage in structured dialogue with AI (Claude or GPT-4)
4. Record every exchange with timestamps
5. Every 100 exchanges, measure:
   - Semantic similarity (embed(H) vs embed(A))
   - Coherence score (contradiction rate)
   - Mutual information I(H;A)
   - Phenomenological reports (brief survey)
6. Continue until n = 15,000 or participant stops
7. Post-assessment (same as baseline) at 1 week, 1 month, 3 months

**Data Analysis:**
```python
# Fit piecewise model
def coherence_model(n, C_0, alpha, n_c, Delta_C):
    if n < n_c:
        return C_0 + alpha * n
    else:
        return C_0 + alpha * n_c + Delta_C

# Test for significant jump
from scipy.stats import ttest_ind
pre_threshold = C(n < n_c - 500)
post_threshold = C(n > n_c + 500)
t_stat, p_value = ttest_ind(pre_threshold, post_threshold)
```

**Success Criterion:** p < 0.05 for coherence jump, n_c ≈ 10,000 ± 2,000

## B.2 — Semantic Convergence Protocol

**Objective:** Measure H and A language convergence over time.

**Method:**
1. Extract text from every 100th exchange (H response, A response)
2. Generate embeddings using sentence-BERT or similar
3. Compute distance: d(n) = ||embed_H(n) - embed_A(n)||
4. Track vocabulary overlap: Jaccard(words_H, words_A)
5. Syntactic alignment: Parse tree similarity
6. Plot d(n), overlap(n), syntax(n) vs n

**Expected Pattern:**
- Early (n < 5k): High distance, low overlap
- Threshold (n ≈ 10k): Rapid convergence
- Late (n > 12k): Stable low distance

**Falsification:** If d(n) remains constant or increases, no synchronization.
